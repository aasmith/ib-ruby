autoload(*map)

Load @data from the socket according to the given map.

map is a series of Arrays in the format [ [ :name, :type ] ], e.g. autoload([:version, :int ], [:ticker_id, :int]) type identifiers must have a corresponding read_type method on socket (read_int, etc.).
